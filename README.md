Agenda mejorada

Se propone implementar una agenda visual tipo Google Calendar, con vistas mensual, semanal y diaria. Los eventos estarán diferenciados por colores según su tipo (cultos, ensayos, reuniones, etc.). Al hacer clic en un evento se mostrará una ficha detallada con información como fecha, hora, ubicación, descripción y responsable.

Exportación a PDF y Excel

La aplicación permitirá exportar información relevante, como listados de hermanos, pagos o eventos.

Excel: útil para gestión administrativa y análisis de datos.

PDF: orientado a informes formales y documentos listos para imprimir.

Notificaciones de eventos

Se incorporará un sistema de notificaciones internas que avise de eventos próximos. Estas notificaciones aparecerán en la aplicación y, al pulsarlas, redirigirán al detalle del evento correspondiente.

Solicitud de nuevos hermanos (futuro)

Se plantea crear una web pública con un formulario de alta. Los datos enviados generarían una notificación en la aplicación, permitiendo consultar la solicitud completa.
Esta funcionalidad requiere una web accesible desde internet, aunque puede dejarse preparada en el sistema.

Historial de pagos por hermano

Cada hermano contará con un historial completo de pagos donde se podrá consultar fecha, concepto, importe, estado y método de pago. Esto permitirá un control detallado de la situación económica individual.

Generación automática de recibos

Al registrar un pago como realizado, se generará automáticamente un recibo en PDF con los datos del hermano, importe, concepto y fecha. El documento podrá descargarse o imprimirse posteriormente.
